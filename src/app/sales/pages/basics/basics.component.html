Date | Fecha | Pipe

This pipe is used to show dates.

  • date
  • date | date
  • date | date: 'short'
  • date | date: 'medium'
  • date | date: 'MMMM DD, YYYY'
  • date | date:'long': spanish
  • date | date:'long': french
  • {{ date }}
  • {{ date | date }}
  • {{ date | date:'short' }}
  • {{ date | date:'medium' }}
  • {{ date | date: 'MMMM dd, YYYY' }}
  • {{ date | date:'long': 'GMT-10' : 'es' }}
  • {{ date | date:'long': 'GMT-10' : 'fr' }}
